### 补丁基础信息
- 版本：KA_INSTALL_PATCH_230603
- 用途：可选补丁，用于兼容老版本教程的vnc指令。

### 注意事项
之前安装过补丁的或者已经安装其他vnc的用户不要安装。

### 如何安装
1. 打开Termux，并进入kali系统。
2. 运行以下命令

```bash
wget https://res.hestudio.net/kali_for_android/patch/ka_install_patch_230603 && bash ka_install_patch_230603
```
